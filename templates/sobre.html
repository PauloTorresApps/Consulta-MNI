Sobre o Sistema

Sistema de Consulta SOAP para MNI

Descrição

Sistema desenvolvido para facilitar consultas de processos judiciais através do MNI (Modelo Nacional de Interoperabilidade), utilizando SOAP/WSDL.

O MNI é um padrão do CNJ para comunicação entre sistemas judiciais no Brasil.

Tecnologias

Python 3 Linguagem principal
Flask Framework web
Zeep Cliente SOAP/WSDL
LXML Processamento XML
Tabler Interface UI
JavaScript Frontend interativo

API REST

POST /api/consultar

Endpoint para consultas programáticas

Exemplo de requisição: 
{
  "numero_processo": "00058128320258272729",
  "incluir_movimentos": true,
  "incluir_documentos": true
}
Resposta de sucesso: 
{
  "success": true,
  "data": { /* dados do processo */ }
}

Funcionalidades

Consulta por número CNJ
Filtros por data
Opções configuráveis
Visualização organizada
Exportação JSON
Debug XML SOAP
API REST integrada
Interface responsiva

Segurança

  • Credenciais em variáveis de ambiente
  • Validação de dados de entrada
  • Tratamento de erros
  • Logs de auditoria

Aviso Legal

Esta é uma ferramenta de consulta auxiliar. Não substitui o acesso oficial aos sistemas do Poder Judiciário. Use apenas com autorização e credenciais válidas.
